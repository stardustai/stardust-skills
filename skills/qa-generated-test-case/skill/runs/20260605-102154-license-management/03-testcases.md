## 待确认问题

1. license 管理列表最终字段和列顺序需确认：PRD 表头只列出序号、类别、项目名称、状态、席位数、标注数量、申请时间、有效期、License 内容、备注，但正文还定义了申请码、域名、生效时间、创建人、操作。
2. 状态判断口径需确认：PRD 写“当前时间与申请时间对比”，但业务上可能应按有效期或到期时间判断。
3. 项目名称、类别、状态搜索是否支持组合搜索，PRD 未说明。
4. 复制 License、复制申请码成功后是否需要提示，以及提示文案需确认。
5. 非 Admin 授权到期登录时的拦截页面和提示文案需确认。
6. 空 License、非法 License、环境不匹配 License 的错误提示文案需确认。
7. 私有化部署“规定期限外”的判断来源、后端字段和时区口径需确认。
