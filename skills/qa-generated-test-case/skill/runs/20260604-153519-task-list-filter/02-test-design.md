# 任务列表页新增筛选，数据展示功能 - 测试设计

## 1. 测试目标

验证任务列表页新增筛选、数量展示、跨页选中、导出选中任务、时间字段展示等能力符合 PRD 要求，并覆盖大数据量、分页、状态保留、边界数量、历史数据兼容和页面数据一致性风险。

## 2. 覆盖分层

| 层级 | 覆盖重点 |
| --- | --- |
| 页面交互 | 筛选面板弹出、添加条件、切换维度、删除条件、关闭标签、数量文案展示 |
| 数据查询 | 多条件 AND、输入或删除条件触发查询、筛选命中总量、无结果 |
| 选择状态 | 单选、取消、选择全部、跨页累计、返回上一页状态保留、清空 |
| 批量操作 | 导出选中任务仅作用于实际选中集合 |
| 分页性能 | 新增 500 条/页、1000 条/页，大分页下筛选和勾选可用 |
| 时间展示 | 更新时间绝对时间、分配时间列、未流入标注池展示 `-`、首次流入时间不覆盖 |
| 历史兼容 | 旧任务、未流入标注池任务、重新流转任务展示稳定 |

## 3. 角色-权限矩阵

| 角色 | 页面访问 | 筛选 | 勾选 | 跨页选中 | 导出选中 | 备注 |
| --- | --- | --- | --- | --- | --- | --- |
| 项目管理人员 | 可访问 | 可操作 | 可操作 | 可操作 | 按已有导出权限处理 | PRD 目标角色 |
| 无任务列表权限用户 | 待确认 | 待确认 | 待确认 | 待确认 | 待确认 | PRD 未说明新增权限 code |

## 4. 状态流转表

| 状态对象 | 初始状态 | 触发动作 | 目标状态或结果 |
| --- | --- | --- | --- |
| 筛选面板 | 未打开 | 点击【筛选】 | 在按钮下方弹出 |
| 筛选条件 | 无条件 | 点击【+添加条件】 | 默认新增“当前操作人”条件 |
| 筛选结果数 | 不展示 | 输入筛选值 | 展示“筛选 N”，N 为命中总量 |
| 筛选结果数 | 展示中 | 删除全部条件 | 数字隐藏 |
| 已选中数 | 不展示 | 勾选任务 | 展示“已选择 N 条 ×” |
| 已选中数 | 展示中 | 取消勾选或清空 | 数量减少或隐藏 |
| 跨页选中集合 | 已选第一页数据 | 翻页后继续选中 | 累计所有页选中任务 |
| 分配时间 | 首次流入标注池已记录 | 后续退回或重新流转 | 保持首次流入时间不变 |

## 5. 关键业务规则转测试规则

| 测试规则 | 来源规则 | 风险 |
| --- | --- | --- |
| 多筛选条件必须按 AND 返回交集 | PRD 功能 1 | 返回并集导致任务范围过大 |
| 筛选结果数必须是命中总量 | PRD 功能 2 | 误展示当前页数量 |
| 已选中数必须和筛选结果数独立 | PRD 功能 2 | 两个数量互相覆盖或混算 |
| 跨页选中必须累计且保持选中态 | PRD 功能 3 | 翻页丢失选择状态 |
| 导出范围必须是实际选中集合 | 历史业务记忆补充 | 导出当前页或全部筛选结果 |
| 更新时间必须展示绝对时间 | PRD 功能 4 | 仍展示相对时间 |
| 分配时间必须只记录首次流入标注池 | PRD 功能 4 | 后续流转覆盖首次时间 |

## 6. 高风险点

1. 筛选结果数误用当前页数量，而不是接口总数。
2. 多条件筛选被实现成 OR 或仅最后一个条件生效。
3. 勾选状态只保存在当前页，翻页后丢失。
4. 筛选数量和已选数量同时展示时样式或数据互相覆盖。
5. 导出选中任务误导出当前页、全部筛选结果或历史已选残留数据。
6. 1000 条/页下渲染、勾选、筛选和操作列交互卡顿或异常。
7. 分配时间取最新流转时间，而不是首次进入标注池时间。
8. 历史任务没有分配时间时空值展示不一致。

## 7. 测试数据建议

1. 准备至少 1200 条任务，覆盖分页 500 和 1000 条/页。
2. 准备任务ID、标题、是否优先、数据集、当前操作人、当前操作团队、工作池、任务状态可区分的数据。
3. 准备多条件交集有结果、无结果、跨页有结果的数据。
4. 准备已进入标注池、从未进入标注池、退回后重新进入标注池、重新流转的任务。
5. 准备可跨页选中的任务集合，并校验导出文件中的任务ID集合。
6. 准备更新时间距离当前日期不同的任务，确保不展示相对时间。

## 8. 待确认问题

1. “名称、单位”与详细字段的映射关系。
2. 重复筛选条件、空筛选值、筛选面板关闭行为。
3. 跨页选中后变更筛选条件时选中集合的保留或清空规则。
4. 无已选任务时导出选中任务入口状态。
5. 是否新增筛选、导出或分配时间字段的权限控制。
6. 时间展示时区和后端字段来源。

## 9. 覆盖矩阵

| 覆盖类别 | 测试点编号 | 关联功能点 | 风险说明 |
| --- | --- | --- | --- |
| 功能主流程 | TP-01 | 打开筛选、添加条件、输入值筛选 | 用户无法完成基础筛选 |
| 分支流程 | TP-02 | 切换不同筛选维度和控件 | 字段控件映射错误 |
| 异常场景 | TP-03 | 无结果、删除条件、清空选择 | 页面状态未刷新 |
| 边界值 | TP-04 | 0 条、500 条、1000 条、跨页累计 | 数量和分页边界错误 |
| 权限依赖 | TP-05 | 已有任务列表权限、导出权限 | 越权或入口不可用 |
| 上下游依赖 | TP-06 | 查询接口总数、导出接口、任务流转时间 | 后端字段口径错误 |
| 历史兼容 | TP-07 | 历史任务、未流入标注池任务 | 空值或旧数据展示异常 |
